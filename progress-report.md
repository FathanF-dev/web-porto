# Personal Portfolio Website – Progress Report (50% Process)

## 1. Project Overview
The project is to build a personal portfolio website inspired by modern design trends. The website serves as a centralized hub to showcase my academic background, skills, projects, and contact information. Currently, the project is at the 50% completion mark, where substantial planning, UI/UX conceptualization, and initial project scaffolding have been completed.

**Tools/Technologies Planned and Used:**
- **Frontend Framework**: React.js
- **Build Tool**: Vite (for rapid development and optimized builds)
- **Styling**: Vanilla CSS with CSS Variables for Dark/Light mode theming
- **Icons**: React Icons (or FontAwesome)
- **Version Control**: Git & GitHub

## 2. Website Structure & Sitemap
The website follows a single-page application (SPA) model with a fixed, center-bottom navigation bar for quick access to all sections.

**Basic Navigation Flow (Sitemap):**
1. **Home**: Landing section with a brief introduction and an eye-catching hero image.
2. **About Me**: Detailed background on personal and academic history.
3. **Skills**: A visual representation of technical and soft skills.
4. **Portfolio (Projects)**: A gallery showcasing selected works and ongoing projects.
5. **Contact**: Contact details, social links, and email outreach.

The UI has been conceptualized with a specific focus on modern aesthetics:

- **Hero Section (Home)**: Large, bold typography introducing the name and title, with a profile picture adjacent to the text.
- **Color Palette**:

## 6. Deployment Planning
- **Platform**: Vercel
- **Why**: Vercel offers seamless integration with Vite and React applications, providing continuous deployment directly from the GitHub repository.
- **Process**: Once development is finalized locally, the code will be pushed to a GitHub repository, and Vercel will automatically trigger a build and deploy the site to a live URL.
